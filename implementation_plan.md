# Roystar - Premium Musical Instruments E-Commerce

## Plan

### Stack
- Next.js 15 (App Router) + TypeScript
- Tailwind CSS
- Google Fonts (Cabin)

### Structure
```
roystar/
├── app/
│   ├── layout.tsx
│   ├── page.tsx
│   └── globals.css
├── components/
│   ├── Navbar.tsx
│   ├── Hero.tsx
│   ├── FeaturedProducts.tsx
│   ├── About.tsx
│   └── Footer.tsx
├── lib/
│   └── products.ts
├── types/
│   └── index.ts
└── public/images/
```

### Pages / Sections
1. Navbar (logo, links, cart icon)
2. Hero (headline, subtitle, CTA, placeholder image)
3. Featured Products (4 cards)
4. About section
5. Footer (links, social, copyright)
